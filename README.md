# Time Management platform

## Project Purpose & Goals 

### About this project

Time Management Portal is a React-based time-tracking tool that lets you log hours manually by project, view a weekly totals, and view clear, one-click reports and instant insights.

### Primary Goal

Deliver a small React App that supports manual time tracking, basic reporting, and JWT-based authentication for a single user account.

## Requirements

### Main features:

JWT-Based Authentication: Simple email/password login keeps data private; tokens refresh silently for uninterrupted sessions.
Project Management: Create and edit projects so every entry is neatly organised for later reporting.
Quick Manual Time Entry: Add hours and seconds with a compact form—select project, pick date, enter duration, hit save.
Built-In Reports: See cards (total hours), bar chart and more data on project overview reports.

### Additional Features:

Dark-Mode Toggle: Switch themes instantly—ideal for late-night logging.
Export: Generate a report for the projects to export. 

**Weekly Overview Calendar:**  Seven-day grid highlights daily totals and gaps, giving you an at-a-glance health check of the week.
